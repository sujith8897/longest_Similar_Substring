# longest_Similar_Substring
### This is the solution for longest similar substring question

#### Example
##### Input
##### arr1 = [3, 6, 7, 4, 2, 7, 9, 1, 0, 3, 5, 6]
##### arr2 = [1, 5, 6, 9, 7, 4, 2, 7, 9, 9, 2, 1]
##### Ouput
##### (7,4,2,7,9)
